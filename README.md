# hbcf-social

Public assets for Hitchin Beer Festival social media — images and shareable media intended to be linked from posts and external sites.

## Layout

- [images/](images/) — image assets. Link directly via `https://raw.githubusercontent.com/lekman/hbcf-social/main/images/<filename>` or via the jsDelivr CDN: `https://cdn.jsdelivr.net/gh/lekman/hbcf-social@main/images/<filename>`.

## Usage

Anything committed here is public. Do not add files containing personal data, internal drafts, or anything you wouldn't post publicly.
